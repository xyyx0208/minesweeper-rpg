### 扫雷→战场核心映射

| 扫雷操作 | 战场效果 |
|---------|---------|
| 翻数字 N | 送 N 个 type=N 兵种到战场 + 生成对应怪物 |
| 插旗数字格 | 该格兵种品质+1阶（白→绿→蓝→紫→橙） |
| 插旗雷格 | 雷变奖励（等级×10 金币） |
| 正确标雷 | 直接击杀当前怪物 / 送 2 精英兵 |
| 错误标雷 | 损失 2 名士兵 |
| Flood-Fill 连消≥3 | 额外送 chain 数士兵 + 前线推进 |
| 踩雷 | 揭示全盘雷位 + Boss 在战场登场 |
| 清空全盘 | 胜利！ |

### 装备系统（6 槽位）

| 槽位 | 属性 | 说明 |
|------|------|------|
| 武器 | ATK + 特殊效果 | 铁剑→龙骨大剑 |
| 防具 | DEF + 特殊效果 | 皮甲→龙鳞甲 |
| 头盔 | DEF + 特殊效果 | — |
| 靴子 | 速度 | — |
| 项链 | 特殊效果 | HP恢复/Gold加成 |
| 神器 | 扫雷联动技能 | 3次安全揭示/显雷方向 |

品质：白→绿→蓝→紫→金→红（同名喂食升阶）
强化：+1~15（+11后降概率，3败保底）
精炼：洗随机属性 | 附魔：镶嵌符文石

**扫雷联动**：神器槽限定特殊次数，如「苍穹锤→3次无风险翻开」

### 技能系统

- **主动技能**：每英雄2-3个主动技，与扫雷动作联动
  - 灼地：连挖3雷后对3×3范围AOE
  - 冻结：选1格下回猜错不炸
  - 相位：最后推理出的数字邻居全翻开
- **被动/光环**：天赋树节点，种族光环（3亡灵→旗+2）
- **扫雷联动**：翻5连触发终结技、推理格回蓝、踩雷触发增伤Buff

### 怪物体系（6 阵营 / 18 种 + 5 Boss）

**阵营**：💀亡灵(死骑/女妖/骨巨人) / ⚙️机械(哨兵/无人机/线圈) / 🐺野兽(霜狼/岩蝎/蛇) / 🔥元素(火灵/水灵/土灵) / 🧑人类(斥候/刺客/战术家) / 👿恶魔(小鬼/地狱犬/领主)

**5 大 Boss**：
1. 📁 **档案员** — 封印棋盘5×5区域，需解外围解锁
2. 🍽️ **暴食者** — 翻安全格则Boss叠加攻击力
3. 💣 **炸弹王** — 每回合在空地埋新雷，棋盘饱和即团灭
4. 👑 **巫妖王** — 数字线索随机错位，逼迫交叉验证
5. 🌌 **渊皇** — P1正常扫 → P2锁为5×5满雷，限时一命通关

### 闯关类型

| 类型 | 说明 |
|------|------|
| 🎯 **突击战** | 破城门(血2000)，敌兵每关+15%强度 |
| 🛡️ **防守战** | 抗8波敌，第5波后每波+30%敌数 |
| 👑 **Boss战** | 单Boss三段技能(P1普攻→P2召唤→P3狂暴) |
| 🧩 **解谜关** | 12步内翻出指定数字组合，奖励高品质兵种 |
| ♾️ **无尽模式** | 每10波+20%攻+10%血，波数排行榜 |

### 结算系统

四档评分 S(×2.0) / A(×1.5) / B(×1.0) / C(×0.5)：

| 维度 | S条件 | A条件 |
|------|-------|-------|
| 步数 | ≤安全格×1.1 | ≤安全格×1.3 |
| 雷探测率 | 100% | ≥90% |
| 连击 | ≥5次 | ≥3次 |
| 无伤 | ✅ | — |

**3星评价**：1星=通关 / 2星=步数达标+雷探测≥85% / 3星=步数优秀+雷探测100%

### 难度系统（动态融合）

| 阶段 | 棋盘 | 雷密度 | 特殊 |
|------|------|--------|------|
| 前期(1-15关) | 6×6→10×10 | 10%→20% | 纯数字推理 |
| 中期(16-30关) | 12×12→16×16 | 20%→28% | 冻土/熔岩地形 |
| 后期(31+关) | 18×18→20×20 | 28%→35% | 迷雾区域需探路 |
